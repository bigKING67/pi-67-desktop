```text
repository: https://github.com/PeakCode-AI/PeakCode

```

The retained design move is the three-region coding workspace: project/session
navigation, a dominant transcript/composer work plane, and contextual activity
or diagnostics. Electron implementation, web renderer code, telemetry, agent
adapters, and visual assets are not copied.

Any future source or asset port must identify the exact file, commit, license,
local destination, and modification in `THIRD_PARTY_NOTICES.md`.
